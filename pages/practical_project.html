A small package has been created to analyze antibiotic resistance gene dynamics over time and across geographic locations. The package offers tools for loading data, temporal analysis, ARIMA-based forecasting, and geospatial visualization. The link to the package is here: https://github.com/anasta874/GeoResistR.

Here’s an example of how to work with the GeoResistR package: 

library(GeoResistR)
## 
## Attaching package: 'GeoResistR'
## The following object is masked from 'package:base':
## 
##     load
data <- load("./synthetic_resistome_data_3_years_trend.csv")
print(colnames(data))
## [1] "gene_id"   "organism"  "sample_id" "abundance" "date"      "latitude" 
## [7] "longitude"
# Get a list of all unique genes
unique_genes <- unique(data$gene_id)
print(unique_genes)
## [1] "vanA"        "ermB"        "blaCTX-M-15" "mecA"        "tetM"
# Анализ временных трендов и построение графиков для первого гена
data_time <- trend(data)
plot_tr(data_time, "blaCTX-M-15")
## `geom_smooth()` using formula = 'y ~ x'

# Анализ временных трендов для другого гена (например, "tetM")
plot_tr(data_time, "tetM")
## `geom_smooth()` using formula = 'y ~ x'

plot_map(data, "tetM")

# Прогнозирование ARIMA для "blaCTX-M-15" и "tetM"
forecast_result_ctxM <- forecast(data_time, "blaCTX-M-15", h = 10)
## Registered S3 method overwritten by 'quantmod':
##   method            from
##   as.zoo.data.frame zoo
forecast_result_tetM <- forecast(data_time, "tetM", h = 10)

# Построение трендов с прогнозами
plot_fc(data_time, forecast_result_ctxM, "blaCTX-M-15")

plot_fc(data_time, forecast_result_tetM, "tetM")
